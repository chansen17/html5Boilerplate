# html5Boilerplate

Simple CLI project to get a basic website structure ap and running quickly

Simple steps: 

1. Be sure to be in the working directory

optional: change package.json files "bin" object file command to suit your needs. This will run the file already created for you.

2. run the command - sudo npm link 
this command links the CLI to bash alias 

3. Tada. You should no be able to run the "touchSite" or you custom command to bootstrap a basic websites structure. 


Thanks for using this project.
